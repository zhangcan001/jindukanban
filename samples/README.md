# 测试样例说明

当前样例目录用于 v3.4-final-stabilization 本地回归。样例覆盖标准单 Sheet、多 Sheet、异常数据和字段映射测试场景；真实 Excel 文件如不便入库，可按文件名在本机验证。

## 标准单 Sheet 导入样例

- 文件：`sample_progress_a.csv`
- 验证方式：新建项目后上传，选择 `CSV` Sheet，按推荐字段映射校验、确认导入并发布。

## 多 Sheet 导入样例

- 文件：真实 Excel `进度导入模板-汽车科技总部产业园-2026-05-12.xlsx`
- 推荐 Sheet：`机电单位`、`消防单位`、`智能化单位`
- 验证方式：在多 Sheet 批量导入中同时选择三个 Sheet，确认三个 Sheet 生成独立批次并可分别发布。

## 异常数据样例

- 文件：`sample_progress_abnormal.csv`
- 验证方式：上传后检查导入校验提示，确认空值、异常日期、异常进度等问题以中文信息展示。

## 字段映射测试样例

- 文件：`sample_progress_b.csv`
- 验证方式：上传后检查字段推荐、字段类型、必填字段缺失提示，并可保存/套用字段映射模板。

## 真实 Excel 验证说明

当前仓库未强制包含真实 Excel 文件。若本机存在 `进度导入模板-汽车科技总部产业园-2026-05-12.xlsx`，请在 v3.4 稳定版回归中至少验证 `机电单位`、`消防单位`、`智能化单位` 三个 Sheet 的导入、发布、Dashboard、预警、整改和报表导出。
