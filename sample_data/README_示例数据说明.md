# 示例数据说明

本目录用于 v3.9-demo-onboarding 首次使用体验，不包含真实项目或敏感数据。

## 文件清单

- `01_单Sheet标准进度表.xlsx`：单 Sheet 标准进度表，适合第一次导入。
- `02_多Sheet进度表.xlsx`：多 Sheet 进度表，按楼栋拆分，适合验证多 Sheet 导入。
- `03_异常数据校验样例.xlsx`：包含少量缺失、负数、超范围百分比和日期异常，用于观察导入校验提示。
- `04_合并表头样例.xlsx`：包含合并标题行和分组表头，用于演示表头确认。

## 字段覆盖

示例数据覆盖：楼栋、楼层、专业、系统、工作内容、单位、总工程量、计划完成量、累计完成量、本期完成量、计划完成率、实际完成率、计划开始、计划完成、责任人、施工单位、备注。

## 推荐体验流程

1. 在工作台或项目列表点击“创建示例项目”。
2. 进入导入页后选择本目录中的 `01_单Sheet标准进度表.xlsx`。
3. 确认 Sheet、表头、字段映射和校验结果。
4. 导入并发布批次后查看 Dashboard、预警、整改闭环和报表。
5. 再使用 `02_多Sheet进度表.xlsx` 体验多 Sheet 导入。

## 注意事项

- 示例数据仅用于演示和测试，可随时删除或重新导入。
- 示例项目名称包含“示例”，可被测试项目清理功能识别。
- 这些文件不改变 Excel 导入、字段识别、进度计算或报表导出的核心逻辑。
