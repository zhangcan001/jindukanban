# 快速发布流程

## 快速迭代规则

- 功能版：开发 + 测试 + 简短报告
- 发布版：最终回归 + 文档 + tag
- 无 P0/P1 不拆 rc
- P2 记录，不阻塞
- 每次必须跑 quick_check

当前稳定候选版本：v3.4-final-stabilization。稳定版收尾阶段只做 P0 / P1 / 明确 P2，不新增业务功能。

## 问题等级

P0：

- 系统无法启动
- 数据丢失
- 导入主流程不可用
- 报表核心导出不可用

P1：

- 核心业务功能不可用
- Dashboard 主指标错误
- 整改闭环关键流程不可用
- 多 Sheet 导入不可用

P2：

- 文案问题
- 样式问题
- 非核心按钮体验
- 个别边界提示不完善

## 每版最小验收

必须确认：

- pytest 通过
- npm run build 通过
- 导入流程未破坏
- Dashboard 未破坏
- 报表导出未破坏
- portable 脚本未明显破坏
